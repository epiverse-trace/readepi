# Get the attribute names and ids of the tracked entities

Use this function to get the ids and names of the features collected
about each tracked entities. The ids and names will be used to update
the list of query parameters.

## Usage

``` r
get_entity_attributes(x)
```

## Arguments

- x:

  A list with the tracked entity attributes

## Value

A data frame with two columns representing the tracked entity attributes
IDs and display names.
